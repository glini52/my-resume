КОНТРОЛЬНІ ПИТАННЯ
  1. Які редактори коду є найбільш поширеними серед веброзробників?
    Visual Studio Code, Sublime Text, Atom, Notepad++, Brackets.
  2. Поясніть поняття тегу. З чого складається елемент?
    Тег це команда в HTML, яка вказує браузеру, як відобразити вміст. Елемент складається з відкриваючого тега, вмісту та закриваючого тега.
  3. Для чого призначений елемент head? Які елементи вміщує даний розділ?
    Елемент head містить метадані, заголовки, посилання на стилі, скрипти та інші інформаційні теги, які не відображаються безпосередньо на сторінці.
  4. Для чого призначений елемент body?
    Елемент body містить основний вміст вебсторінки, який відображається у браузері.
  5. Перерахуйте теги для роботи з текстом.
    p, h1 - h6, b, i, u, strong, em, blockquote, pre, code.
  6. Яке призначення тегу div?
    Тег div використовується для групування блоків елементів і застосування стилів або скриптів до групи.
  7. Яке призначення тегу style?
    Тег style використовується для додавання CSS-стилів безпосередньо в HTML документ.
